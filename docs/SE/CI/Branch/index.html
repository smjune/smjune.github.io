        확인이 완료된 change을 운영 브랜치 (main) 으로 merge

- 해당 브랜치로 부터 WebHook 혹은 Polling 을 통해 CI 서버가 빌드 한다.  
- 빌드 후 자동화된 테스트 를 수행한다.

- 게시된 결과에 따라 코드를 수정한다.
